\section{Теоретическая часть}

\subsection{Новый подход к старой задаче}
В рамках предыдущих работ и точного вычисления было показано, что у двухуровневой системы, описываемой Гамильтонианом:
\begin{equation}
    \hat{H} = E_0 \ \hat{a}^{\dag}_{+} \hat{a}_{+} + \omega_0 \ \hat{b}^{\dag}_{-} \hat{b}_{-} +  g \left( \hat{a}^{\dag}_{+} \hat{b}_{-} + \hat{a}_{+} \hat{b}^{\dag}_{-} \right)
\end{equation}
накаченной лазерным когерентным импульсом, в поляризации $H$, с волновой функцией:
\begin{equation}
    \left| \psi_{in} \right> = e^{-\left| \alpha \right|^2 / 2} e^{\alpha \hat{b}^{\dag}_H} \left| 0 \right> = e^{-\left| \alpha \right|^2 / 2} \sum_n \frac{\alpha^n}{\sqrt{n!}}  \left| n_H \right>
\end{equation}
волновая функция выходного состояния, в поляризации $V$, имела вид:
\begin{gather}
    \left| \psi_{out}^V \right> \approx \cos \left( \frac{\theta }{2} \right) \left| 0 \right> + \frac{i \delta - 1}{\sqrt{2}} \sin \left( \frac{\theta}{2} \right) \left| 1 \right> + \frac{i \delta}{2} \cos \left( \frac{\theta}{2} \right) \left| 2 \right>
\end{gather}
и расчет $g^{(2)}(0)$ на этом состоянии:
\begin{gather}
    g^{(2)}(0) = \frac{ \left< \psi_{out}^V \left| \ \hat{n}^2 - \hat{n} \ \right| \psi_{out}^V \right>}{\left< \psi_{out}^V \left| \ \hat{n} \ \right| \psi_{out}^V \right> ^2 } = \frac{2\left| \delta \right|^2 \cos^2 \left( \frac{\theta}{2} \right) }{\left( \sin^2 \left( \frac{\theta}{2}\right) + \left| \delta \right|^2 \right)^2}
\end{gather}


В рамках предыдущего подхода мы считали взаимодействие всех гармоник когерентного импульса с двухуровневой системой.
Считали эволюцию состояний в обеих поляризациях и лишь в конце отсекали эти состояния детектором.
\begin{gather}
    i \frac{\partial \hat{\rho}_{HV} }{\partial t} = \left[ \hat{H} ; \hat{\rho}_{HV} \right]
\end{gather}

В рамках нового подхода мы в силу большого числа фотонов в поляризации $H \left( N >> 1 \right) $,
мы будем считать это поле классическим и считать динамику будем только в $V$ поляризации:

\par Тогда матрица плотности описывающая систему имеет вид:
\begin{gather}
    \hat{\rho}_{HV} = e^{\alpha \hat{b}^{\dag}_{H}}  \hat{\rho}_{V}  e^{\alpha^{*} \hat{b}_{H}}
\end{gather}

\par Просуммировав по всем состояниям в поляризации $H$ (посчитаем $Tr_{H}$ по этим состояниям),
мы можем писать только эволюцию на состояния с поляризацией $V$.
\begin{gather}
    i \frac{\partial \hat{\rho}_{V} }{\partial t} = \left[ \hat{H}_{V} ; \hat{\rho}_{V} \right]
\end{gather}

\par Тогда Гамильтониан, описывающий эволюцию системы, будет иметь вид:
\begin{gather}
    \hat{H}_V = \Omega_R^{H} \left( \hat{d}^{\dag}_{\uparrow} \hat{a}_{\uparrow}
    + \hat{d}_{\uparrow} \hat{a}^{\dag}_{\uparrow} \right)
    + ig \left( \hat{b}_{V} \hat{d}^{\dag}_{\uparrow} \hat{a}_{\uparrow}
    - \hat{b}^{\dag}_{V} \hat{d}_{\uparrow} \hat{a}^{\dag}_{\uparrow} \right)
\end{gather}
Где: $\Omega_R^{H} $ - частота осцилляций Раби, вызванных классическим $H$- поляризованным полем.
$\Omega_R^{H} \sim g \sqrt{N} $, и так как фотонов в лазерном импульсе много $N >> 1 \Rightarrow \Omega_R^{H} >>  g $,
Тогда фоковское пространоство можно сократить до состояний $ \left| 0 \right> $ и $ \hat{b}^{\dag}_{V} \left| 0 \right> $,
с 0 и 1 фотоном соответсвенно.

\par Тогда базисные состояния нашей системы это:
\begin{gather}
    \hat{a}^{\dag}_{\uparrow} \left| 0 \right> \, ; \, \hat{d}^{\dag}_{\uparrow} \left| 0 \right> \, ; \, \hat{b}^{\dag}_{V} \hat{a}^{\dag}_{\uparrow} \left| 0 \right> \, ; \, \hat{b}^{\dag}_{V} \hat{d}^{\dag}_{\uparrow} \left| 0 \right>
\end{gather}
В этом базисе Гамильтониан будет записан как:
\begin{gather}
    \hat{H} =
\begin{pmatrix}
    0 & \Omega_R & 0 & 0 \\
    \Omega_R & 0 & ig & 0 \\
    0 & -ig & 0 & \Omega_R \\
    0 & 0 & \Omega_R & 0
\end{pmatrix}
\end{gather}
И если выделить в нем блоки по фотонным состояниям:
\begin{gather}
    \hat{H} =
\begin{pmatrix}
    \hat{H_0} & \hat{V} \\
    \hat{V}^{\dag} & \hat{H_0}
\end{pmatrix}
\end{gather}
Теперь если переписать эту задачу в представлении взаимодействия, то получим:
\begin{gather}
    -i \frac{\partial}{\partial t } \left| \chi \right> =
\begin{pmatrix}
    0 &  \hat{V_t} \\
     \hat{V_t}^{\dag} & 0
\end{pmatrix}
    \left| \chi \right> \end{gather}
Где:
\begin{gather}
    \hat{V_t} = e^{i\hat{H_0}t} \hat{V} e^{-i\hat{H_0}t} \\ \nonumber
    \left| \psi \right> =
\begin{pmatrix}
    e^{-i\hat{H_0}t} & 0 \\
    0 & e^{-i\hat{H_0}t}
\end{pmatrix}
    \left| \chi \right> \\ \nonumber
    \left| \chi(0) \right> = \left| \psi(0) \right> = \hat{a}^{\dag}_{\uparrow} \left| 0 \right>
\end{gather}
Решая эту задачу по теории возмущений в первом порядке по $g$, получаем ответ на $\left| \chi \right>$:
\begin{gather}
    \left| \chi \right> \approx \left( \hat{I} -i \int^t_0 d\tau
\begin{pmatrix}
    0 &  \hat{V_t} \\
     \hat{V_t}^{\dag} & 0
\end{pmatrix}
    \right) \left| \chi(0) \right>
\end{gather}
Так как:
\begin{gather}
    -i \int^t_0 d\tau \hat{V_t}
\approx
\frac{gt}{2}
\begin{pmatrix}
    0 & 1 \\
    1 & 0
\end{pmatrix}
\end{gather}
То можно получить финальное выражение для $\left| \psi_{out} \right>$
\begin{gather}
    \left| \psi_{out} \right> = \cos(\Omega_R t) \hat{a}^{\dag}_{\uparrow} \left| 0 \right>  -i\sin(\Omega_R t) \hat{d}^{\dag}_{\uparrow} \left| 0 \right> + gt \, i\sin(\Omega_R t) \hat{a}^{\dag}_{\uparrow} \hat{b}^{\dag}_{V} \left| 0 \right> - \\ \nonumber
    - gt \, \cos(\Omega_R t) \hat{d}^{\dag}_{\uparrow} \hat{b}^{\dag}_{V} \left| 0 \right>
\end{gather}
После того как трион высвечивается, а на детекторе мы регистрируем фотоны в $V$-поляризации, то:
\begin{gather}
    \left| \psi_{out} \right> = \cos(\Omega_R t) \left| 0 \right>  + \frac{i gt -1}{\sqrt{2}} \sin(\Omega_R t) \left| 1 \right> + \frac{i gt}{2} \, \cos(\Omega_R t)  \left| 2 \right>
\end{gather}

Как видим, этот ответ полностью совпадает с ответом полученным в [1],
и поэтому мы будем использовать этот подход дальше в задачах.

\subsection{Новый подход}
\par В рамках нового подхода будем возбуждать систему нерезонансным лазерным возмущением.
Это позволит отсекать выходное состояние по частотам в узком диапазоне, без лазерной засветки,
во всех поляризациях.

Такая система будет теперь описываться Гамильтонианом учитывающим обе поляризации излучения, а так же взаимодействие с магнитным полем.
\begin{gather}
    \hat{H}_{+} = \Delta \hat{d}^{\dag}_{\uparrow} \hat{d}_{\uparrow} + \Delta \hat{b}^{\dag}_{+} \hat{b}_{+} + \Omega_R \left( \hat{d}^{\dag}_{\uparrow} \hat{a}_{\uparrow}
    +  \hat{d}_{\uparrow} \hat{a}^{\dag}_{\uparrow} \right) + g \left( \hat{b}_{+} \hat{d}^{\dag}_{\uparrow} \hat{a}_{\uparrow}
    +  \hat{b}^{\dag}_{+} \hat{d}_{\uparrow} \hat{a}^{\dag}_{\uparrow} \right) \\ \nonumber
    \hat{H}_{-} = \Delta \hat{d}^{\dag}_{\downarrow} \hat{d}_{\downarrow} + \Delta \hat{b}^{\dag}_{-} \hat{b}_{-} + \Omega_R \left( \hat{d}^{\dag}_{\downarrow} \hat{a}_{\downarrow}
    +  \hat{d}_{\downarrow} \hat{a}^{\dag}_{\downarrow} \right) + g \left( \hat{b}_{-} \hat{d}^{\dag}_{\downarrow} \hat{a}_{\downarrow}
    +  \hat{b}^{\dag}_{-} \hat{d}_{\downarrow} \hat{a}^{\dag}_{\downarrow} \right) \\ \nonumber
    \hat{H}_{B} = \alpha B_x \left( \hat{a}^{\dag}_{\downarrow} \hat{a}_{\uparrow} + \hat{a}^{\dag}_{\uparrow} \hat{a}_{\downarrow}  \right) + \alpha B_z \left( \hat{a}^{\dag}_{\uparrow} \hat{a}_{\uparrow} - \hat{a}^{\dag}_{\downarrow} \hat{a}_{\downarrow}  \right) + \beta B_z \left( \hat{d}^{\dag}_{\uparrow} \hat{d}_{\uparrow} - \hat{d}^{\dag}_{\downarrow} \hat{d}_{\downarrow}  \right) \\ \nonumber
    \hat{H} = \hat{H}_{+} + \hat{H}_{-} + \hat{H}_{B} \nonumber
\end{gather}
