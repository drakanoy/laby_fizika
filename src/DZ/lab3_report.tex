\par \textbf{Задача}:
\par Исследовать методы оптимизации нулевого, первого и второго порядка на наборе из трёх двумерных функций. Для каждой функции требуется найти минимум из нескольких начальных точек, построить траектории, сравнить методы по точности и вычислительным затратам, а также провести автоматический подбор гиперпараметров через библиотеку \texttt{Optuna}.

\par В сравнение включены четыре метода:
\begin{itemize}
    \item \texttt{Nelder-Mead} --- метод нулевого порядка;
    \item \texttt{CG} --- библиотечный метод первого порядка из \texttt{scipy.optimize.minimize};
    \item \texttt{Newton-CG} --- метод второго порядка;
    \item \texttt{Gradient Descent} --- самописный градиентный спуск
\end{itemize}

\par \textbf{1. Описание реализации}

\par В лабораторной работе реализованы три тестовые функции двух переменных, аналитические формулы для градиентов и гессианов, единая обёртка над \texttt{scipy.optimize.minimize}, самописный градиентный спуск с убывающим шагом и backtracking по условию Армихо, а также блок автоматического подбора гиперпараметров с помощью \texttt{Optuna}.

\par Для каждого запуска сохранялись следующие метрики:
\begin{itemize}

    \item значение функции в этой точке;
    \item число итераций;
    \item число вызовов функции;
    \item число вызовов градиента;
    \item число вызовов гессиана;
    \item расстояние до ближайшего точного минимума.
\end{itemize}

\par \textbf{2. Теоретическая справка}

\par Рассматривается задача безусловной минимизации
\[
 x^* = \arg\min_{x \in \mathbb{R}^2} f(x).
\]
Методы нулевого порядка используют только значения функции, методы первого порядка используют градиент, а методы второго порядка дополнительно учитывают кривизну поверхности через гессиан.

\par Для самописного градиентного спуска использовалась схема
\[
 x_{k+1} = x_k - \alpha_k \nabla f(x_k),
\]
где шаг уменьшался по формуле
\[
\alpha_k = \alpha_0 \gamma^k,
\]
а затем дополнительно корректировался через backtracking. Это важно на функциях с вытянутыми долинами: например, на функции Розенброка без контроля шага метод начинает сильно колебаться. В терминах геометрии это похоже на движение по узкому оврагу: если смотреть только на локальный уклон, можно долго идти зигзагом, не используя информацию о форме долины.

\par В методе Ньютона направление определяется решением системы
\[
H(x_k)p_k = -\nabla f(x_k),
\]
после чего выполняется переход вдоль найденного направления. Метод \texttt{CG} тоже относится к методам первого порядка, но использует сопряжённые направления, поэтому обычно работает заметно быстрее на гладких функциях, чем обычный градиентный спуск.

\par В ходе лабораторной работы были исследованы три функции оптимизации и четыре алгоритма: \texttt{Nelder-Mead}, \texttt{CG}, \texttt{Newton-CG} и самописный \texttt{Gradient Descent}.

На гладких функциях метод \texttt{CG} оказался существенно эффективнее обычного градиентного спуска и во многих случаях приблизился по качеству к методу второго порядка.

\par Основные выводы такие:
\begin{itemize}
    \item \texttt{CG} --- сильный универсальный метод первого порядка: на Розенброке и Химмельблау он дал очень точные решения при умеренной вычислительной цене;
    \item \texttt{Newton-CG} остаётся самым быстрым на хороших стартах, но не застрахован от плохих траекторий на сложной функции Била;
    \item \texttt{Nelder-Mead} не требует производных и хорошо работает на умеренно сложных функциях, однако на Биле может уходить в неудачную область;
    \item самописный \texttt{Gradient Descent} оказался полезен как дополнительный эксперимент: он наглядно показывает, насколько сильно качество метода первого порядка зависит от выбора шага и от геометрии поверхности;
    \item подбор гиперпараметров через \texttt{Optuna} помогает ускорять методы, но это ускорение нередко достигается ценой небольшого ухудшения точности, если оптимизируется составной критерий.
\end{itemize}

%\par Таким образом, после добавления \texttt{CG} основное сравнение стало более полным: теперь в работе представлены метод без производных, библиотечный метод первого порядка, самописный метод первого порядка и метод второго порядка. Это позволяет гораздо нагляднее увидеть, как использование градиента и кривизны влияет на скорость и устойчивость сходимости.
