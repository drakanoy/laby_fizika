\par \textbf{Задача}:
\par Исследовать прямые методы решения систем линейных алгебраических уравнений:
\begin{enumerate}
    \item сгенерировать квадратную вещественную матрицу $A$ и вектор правой части $b$ размерности $n=6$;
    \item реализовать простой метод Гаусса, метод Гаусса с выбором ведущего элемента по столбцу и метод Гаусса с выбором ведущего элемента по всей матрице;
    \item сравнить полученные решения с решением, найденным библиотечными средствами;
    \item найти обратную матрицу методом Гаусса;
    \item сравнить найденную обратную матрицу с библиотечной;
    \item вычислить число обусловленности в $1$-норме, $2$-норме и бесконечной норме;
    \item выполнить дополнительное исследование, демонстрирующее неустойчивость простого метода Гаусса при неудачном выборе ведущего элемента.
\end{enumerate}

Основная часть лабораторной посвящена сравнению трёх вариантов метода Гаусса и библиотечных алгоритмов. В дополнительной части необходимо подобрать пример, в котором простой метод Гаусса и метод с выбором ведущего элемента дают существенно разные результаты, а также экспериментально подтвердить выводы о чувствительности системы к малым возмущениям.

\par \textbf{Решение}:

\par \textbf{1. Описание реализации}

\par В программе на Python были реализованы следующие функции:
\begin{itemize}
    \item \texttt{gauss(a, b)} --- простой метод Гаусса без выбора ведущего элемента;
    \item \texttt{gauss\_pivot(a, b)} --- метод Гаусса с выбором ведущего элемента по столбцу;
    \item \texttt{gauss\_full\_pivot(a, b)} --- метод Гаусса с выбором ведущего элемента по всей оставшейся подматрице;
    \item \texttt{inverse\_gauss(a)} --- вычисление обратной матрицы методом Гаусса--Жордана с выбором ведущего элемента по столбцу.
\end{itemize}

\par Для проверки правильности результатов использовались функции библиотеки \texttt{NumPy}:
\[
\texttt{numpy.linalg.solve}, \qquad \texttt{numpy.linalg.inv}, \qquad \texttt{numpy.linalg.norm}.
\]

\par \textbf{2. Теоретическая справка}

\par Для системы линейных уравнений
\[
Ax=b
\]
метод Гаусса состоит из двух этапов:
\begin{enumerate}
    \item прямой ход --- приведение матрицы к верхнетреугольному виду;
    \item обратный ход --- последовательное нахождение неизвестных, начиная с последнего уравнения.
\end{enumerate}

\par Если на очередном шаге ведущий элемент мал по модулю, то деление на него может усилить вычислительные погрешности. Поэтому используются модификации метода:
\begin{itemize}
    \item выбор ведущего элемента по столбцу;
    \item выбор ведущего элемента по всей матрице.
\end{itemize}

\par Число обусловленности матрицы в норме $\|\cdot\|$ определяется формулой
\[
\operatorname{cond}(A)=\|A\|\cdot\|A^{-1}\|.
\]
Чем больше это число, тем чувствительнее решение системы к малым возмущениям исходных данных.

\[

\]

\par Главная идея примера состоит в том, что первый ведущий элемент равен $10^{-14}$, то есть чрезвычайно мал. В простом методе Гаусса это приводит к делению на почти нулевой элемент и накоплению больших ошибок округления. При использовании выбора ведущего элемента по столбцу алгоритм сначала переставляет строки, поэтому в качестве ведущего выбирается элемент порядка единицы, и вычисления становятся устойчивыми.

\par \textbf{5. Анализ результатов}

\begin{enumerate}
    \item Для случайно сгенерированной невырожденной матрицы размерности $6\times 6$ все методы дали совпадающие результаты с точностью до машинного нуля.
    \item Реализация нахождения обратной матрицы методом Гаусса показала полное совпадение с библиотечной функцией.
    \item Числа обусловленности исходной матрицы оказались сравнительно небольшими, поэтому система является устойчивой к малым погрешностям.
    \item На специально построенном примере было показано, что простой метод Гаусса без выбора ведущего элемента может давать существенно худший результат даже тогда, когда сама матрица не является сильно плохо обусловленной.
    \item Основной источник ошибки --- деление на очень малый ведущий элемент на первом шаге исключения.
    \item Метод Гаусса с выбором ведущего элемента по столбцу и библиотечный алгоритм оказались устойчивыми и восстановили точное решение с точностью до машинной.
\end{enumerate}

\par \textbf{Вывод}

\par В ходе лабораторной работы были реализованы и исследованы прямые методы решения СЛАУ. Для обычных хорошо обусловленных систем все рассмотренные варианты метода Гаусса дают практически одинаковые результаты. Однако дополнительный эксперимент показал, что отказ от выбора ведущего элемента может привести к заметной потере точности даже при умеренном числе обусловленности матрицы.

\par Следовательно, при практическом решении систем линейных уравнений предпочтительно использовать модифицированные варианты метода Гаусса с выбором ведущего элемента, так как они существенно устойчивее к накоплению вычислительных погрешностей.
