\par Условие:
\par Найти Гамильтониан орбитального взаимодействия (линейный по $K$) для группы $T_d$.
\par

\par \textbf{Решение}:
%\par
%\includegraphics[width=1\textwidth]{photo_11.jpg}
%\par
\par В самом общем случае Гамильтониан выглядит вот так:
\[
    H_{so} = C_1 J_{\alpha} K_{\beta} + C_2 J_{\alpha} J_{\beta} K_{\gamma} + C_3 J_{\alpha} J_{\beta} J_{\gamma} K_{\delta}
\]
\par Слагаемого с $C_1$ не может быть так как их произведение не содержит скалярного неприводимого представления $A_1$, а слагаемого с $C_2$ не может быть по инверсии времени:
\[
    H_{so} = C_3 J_{\alpha} J_{\beta} J_{\gamma} K_{\delta}
\]
\par Теперь рассмотрим как раскладывается представление с $C_3$:
\[
    F_1 \otimes F_1 \otimes F_1 \otimes F_2 = (F_1 \otimes F_1) \otimes (F_1 \otimes F_2) = (A_1 \oplus E \oplus F_1 \oplus F_2) \otimes (A_2 \oplus E \oplus F_1 \oplus F_2) =
\]
\par Нам интересны только $(E \oplus E)$, $(F_1 \oplus F_1$ и $(F_2 \oplus F_2)$, потому что только эти умножения содержат представление $A_1$. Распишим их:
\[
    E \otimes E = \left[ \sqrt{3} (J_x^2 - J_y^2) + (2J_z^2 - J_x^2 - J_y^2) \right] \left[ \sqrt{3} (J_x K_x - J_y K_y) + (2J_z K_z - J_x K_x - J_y K_y) \right]
\]
\par Выделяем представление $A_1$:
\[
    \Rightarrow 3 (J_x^2 - J_y^2) (J_x K_x - J_y K_y) + (2J_z^2 - J_x^2 - J_y^2) (2J_z K_z - J_x K_x - J_y K_y)
\]
\[
    F_1 \otimes F_1 = \left[ \vec{J} \times \vec{J} \right] \left[ \vec{J} \times \vec{K} \right] \Rightarrow 0
\]
\[
    F_2 \otimes F_2 = \left[ \{ J_x ; J_y \}_s + \{ J_y ; J_z \}_s + \{ J_z ; J_x \}_s \right] \left[ J_x K_y + J_y K_z+ J_z K_x \right]
\]
\[
    \Rightarrow \{ J_x ; J_y \}_s J_x K_y + \{ J_y ; J_z \}_s J_y K_z + \{ J_z ; J_x \}_s J_z K_x
\]
\par Таким образом Гамильтониан:
\[
    H_{so} = A_3 \left[ 3 (J_x^2 - J_y^2) (J_x K_x - J_y K_y) + (2J_z^2 - J_x^2 - J_y^2) (2J_z K_z - J_x K_x - J_y K_y) + \right.
\]
\[
    \left. + \{ J_x ; J_y \}_s J_x K_y + \{ J_y ; J_z \}_s J_y K_z + \{ J_z ; J_x \}_s J_z K_x \right]
\]
