\par Условие:
\par Найти тензор связи между $\varepsilon_{\alpha \beta}$ и $u_{\alpha \beta}$ в группе $T_d$ из соображений симметрии.

\par \textbf{Решение}:
%\par
%\includegraphics[width=1\textwidth]{photo_11.jpg}
%\par
\par Оба тензора преобразуются по представлению $F_2 \otimes F_2$:
\[
    F_2 \otimes F_2 = (A_1 \oplus E \oplus F_1 \oplus F_2)
\]
\par Так как $\varepsilon_{\alpha \beta} = \varepsilon_{ \beta \alpha}$ и $u_{\alpha \beta} = u_{ \beta \alpha}$ то $F_1$ обнуляется.
\[
    A_1 \oplus E \oplus F_2
\]
\par Величины с одинаковым представлением должны быть пропорциональны:
\[
    A_1: Tr\hat{\varepsilon} = C_1 Tr\hat{u}
\]
\[
    E: \{ \sqrt{3} \left( \varepsilon_{xx} - \varepsilon_{yy} \right) ; 2\varepsilon_{zz} - \varepsilon_{xx} - \varepsilon_{yy} \} = C_2 \{ \sqrt{3} \left( u_{xx} - u_{yy} \right) ; 2u_{zz} - u_{xx} - u_{yy} \}
\]
\[
    F_2: \{ \varepsilon_{yz} ; \varepsilon_{zx} ; \varepsilon_{xy} \} = C_3 \{ u_{yz} ; u_{zx} ; u_{xy} \}
\]
\par Тогда тензор связывающий их выглядит вот так:
\[
    \varepsilon_{\alpha \beta } = a_{\alpha \beta \gamma \delta } u_{\gamma \delta }
\]
\[
    a_{\alpha \beta \gamma \delta } = C_1 \delta_{\alpha \beta} \delta_{\gamma \delta} \frac{1}{3} + C_2 \delta_{\alpha \beta} \delta_{\gamma \delta} \left( \delta_{\alpha \gamma} - \frac{1}{3} \right) + C_3 \left( 1 - \delta_{\alpha \beta} \right) \delta_{\alpha \gamma} \delta_{\beta \delta}
\]
