\par \textbf{Задача}:
\par Исследовать итерационные методы решения систем линейных алгебраических уравнений и численные методы решения спектральной задачи для симметрической матрицы порядка $n=6$:
\begin{enumerate}
    \item сгенерировать симметричную вещественную матрицу $A$ и вектор правой части $b$;
    \item найти решение системы $Ax=b$ прямым методом из лабораторной работы 1;
    \item найти решение системы методами Якоби и Зейделя с точностью $\varepsilon_1 = 10^{-4}$ и $\varepsilon_2 = 10^{-12}$ в норме $\|\cdot\|_\infty$;
    \item вычислить априорные оценки числа итераций и сравнить их с фактическим числом итераций;
    \item найти наибольшее по модулю собственное число и соответствующий собственный вектор степенным методом и методом скалярных произведений;
    \item найти наименьшее по модулю собственное число и соответствующий собственный вектор методом обратных итераций и методом сдвига;
    \item сравнить найденные собственные числа с результатами библиотечной функции;
    \item вычислить число обусловленности по собственным числам и по нормам матрицы и её обратной, а затем сравнить результаты.
\end{enumerate}

\par \textbf{Решение}:

\par \textbf{1. Описание реализации}

\par
\begin{itemize}
    \item \texttt{gauss\_pivot(a, b)} --- решение системы методом Гаусса с выбором главного элемента по столбцу;
    \item \texttt{inverse\_gauss(a)} --- вычисление обратной матрицы методом Гаусса--Жордана;
    \item \texttt{jacobi(A, b, eps)} --- метод Якоби;
    \item \texttt{seidel(A, b, eps)} --- метод Зейделя;
    \item \texttt{apriori\_steps\_jacobi(A, b, eps)} --- априорная оценка числа итераций метода Якоби;
    \item \texttt{apriori\_steps\_seidel(A, b, eps)} --- априорная оценка числа итераций метода Зейделя;
    \item \texttt{power\_method(A, eps)} --- степенной метод;
    \item \texttt{scalar\_product\_method(A, eps)} --- метод скалярных произведений;
    \item \texttt{inverse\_iteration(A, eps)} --- метод обратных итераций;
    \item \texttt{shift\_method(A, mu, eps)} --- метод сдвига.
\end{itemize}

\par Для обеспечения сходимости итерационных методов матрица строилась симметричной, после чего на диагонали обеспечивалось диагональное преобладание, для сходимости методов:
\[
A = \frac{M + M^T}{2},
\]
после чего
\[
a_{ii} = \sum_{j=1}^{n} |a_{ij}| + 1.

\par \textbf{5. Число обусловленности}

\par Для симметрической матрицы число обусловленности в спектральной норме может быть найдено по формуле
\[
\operatorname{cond}_2(A)=\frac{|\lambda_{\max}|}{|\lambda_{\min}|}.
\]

\par По найденным собственным значениям получено:
\[
\operatorname{cond}_2(A)\approx \frac{51.07980060061792}{12.406500781226194}\approx 4.117180299372815.
\]

\par По определению через нормы матрицы и обратной матрицы:
\[
\operatorname{cond}_2(A)=\|A\|_2\cdot\|A^{-1}\|_2 \approx 4.117180299373134.
\]

\par \textbf{6. Анализ результатов}

\begin{enumerate}
    \item Построенная матрица симметрична и обладает диагональным преобладанием, поэтому методы Якоби и Зейделя сходятся.
    \item Метод Зейделя оказался заметно быстрее метода Якоби: например, при точности $10^{-12}$ ему потребовалось $16$ итераций, тогда как методу Якоби --- $68$.
    \item Априорные оценки в обоих случаях оказались существенно больше фактического числа итераций. Это объясняется тем, что такие оценки являются гарантированными верхними границами и обычно оказываются грубыми.
    \item При увеличении требуемой точности количество итераций возрастает, что соответствует теории итерационных методов.
    \item Методы для спектральной задачи дали собственные числа, совпадающие с библиотечными до очень высокой точности.
    \item Невязки для найденных собственных пар имеют порядок $10^{-6}$, что говорит о хорошем качестве найденных приближений.
    \item Число обусловленности, вычисленное двумя разными способами, совпало, что подтверждает правильность реализации и теоретическую связь между спектром симметрической матрицы и её спектральной нормой.
\end{enumerate}
